# AI Frontend Capstone

My AI Frontend Capstone Project.

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR-USERNAME/ai-frontend-capstone.git
```

2. Go into the project folder:
```bash
cd ai-frontend-capstone
```

3. Install dependencies:
```bash
npm install
```

4. Start the project:
```bash
npm start
```
